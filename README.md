# RabbitMqApp

Repositório de estudos sobre RabbitMQ. O código aqui presente foi baseado no tutorial do site do RabbitMQ: https://www.rabbitmq.com/tutorials/tutorial-one-dotnet.html

Consumer rodando em console application e o publisher em web api.

 - .NET 6.0

RabbitMQ rodando dentro de um container docker utilizando docker-compose (docker-compose.yml)
